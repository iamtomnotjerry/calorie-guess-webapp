## 🚀 Overview

**CalorieGuess** is a premium web application designed to help users track their nutritional intake effortlessly. Leveraging the power of **Google's Gemini AI**, CalorieGuess analyzes food images to provide accurate, localized estimations of calories, macronutrients, and ingredients.

### ✨ Key Features
- **AI Vision Analysis**: Instantly identify food and estimate nutritional values from uploaded images.
- **Localized Accuracy**: Fine-tuned logic for regional dishes and ingredients.
- **Glassmorphism UI**: A stunning, modern interface with smooth animations and responsive layouts.
- **One-Click Tracking**: Simple and intuitive user experience designed for speed.

### 🛠 Tech Stack
- **Framework**: [Next.js](https://nextjs.org/) (App Router)
- **AI Engine**: [Google Gemini API](https://ai.google.dev/)
- **Styling**: [Tailwind CSS](https://tailwindcss.com/)
- **Icons**: Lucide React

### 📦 Getting Started

1. **Clone the repository**:
   ```bash
   git clone https://github.com/iamtomnotjerry/calorie-guess-webapp.git
   ```
2. **Install dependencies**:
   ```bash
   npm install
   ```
3. **Set up Environment Variables**:
   Create a `.env.local` file and add your Gemini API Key:
   ```env
   NEXT_PUBLIC_GEMINI_API_KEY=your_api_key_here
   ```
4. **Run the development server**:
   ```bash
   npm run dev
   ```
